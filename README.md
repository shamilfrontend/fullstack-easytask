# EasyTask - Менеджер задач

Полнофункциональный менеджер задач в стиле Trello с поддержкой канбан-досок, карточек задач и командной работы.

## Технологии

### Frontend
- Vue 3
- Vite
- SCSS
- Element Plus
- Pinia
- Vue Router
- Socket.io Client
- Vue Draggable Plus

### Backend
- Node.js
- Express.js
- MongoDB
- Socket.io
- JWT Authentication
- Multer (file uploads)

## Установка

### Backend

1. Перейдите в директорию backend:
```bash
cd backend
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите MongoDB (если не запущен):
```bash
mongod
```

4. Запустите сервер:
```bash
npm run dev
```

### Frontend

1. Перейдите в директорию frontend:
```bash
cd frontend
```

2. Установите зависимости:
```bash
yarn install
```

3. Запустите dev сервер:
```bash
yarn dev
```

Приложение будет доступно по адресу `http://localhost:5173`
